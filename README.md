# aws-ecommerce-platform
A fully serverless e-commerce platform built on AWS.

## Tech Stack
- AWS Lambda
- AWS API Gateway
- AWS DynamoDB
- AWS S3 + CloudFront
- AWS Cognito
- AWS SQS + Step Functions
- AWS SES
- AWS ElastiCache
- AWS Rekognition
- AWS Personalize
- AWS CloudWatch
